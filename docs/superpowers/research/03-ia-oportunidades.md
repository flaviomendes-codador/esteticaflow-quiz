## Seção 3: Modelos de Negócio Detalhados

### B2B Direto para Clínicas

#### Modelo A — SaaS por Assinatura

| Métrica | Clínica Pequena (<R$50K/mês) | Clínica Média (R$50–200K/mês) | Clínica Grande (>R$200K/mês) |
|---------|------------------------------|-------------------------------|-------------------------------|
| Ticket SaaS tolerado | R$150–400/mês | R$400–1.500/mês | R$1.500–5.000/mês |
| Ciclo de venda | 1–3 semanas | 2–6 semanas | 1–3 meses |
| Decisor | Dono(a) direto | Dono(a) + gerente | Diretoria + financeiro |
| Churn anual estimado | 25–40% | 15–25% | 8–15% |
| LTV estimado | R$2K–6K | R$10K–40K | R$40K–200K |
| Canal preferido | Instagram DM, WhatsApp | LinkedIn, indicação | Consultores especializados |

#### Modelo B — Implementação + Recorrência (modelo atual do EsteticaFlow)

**Vantagens:**
- Setup fee financia aquisição e implantação — CAC se paga no ato
- Cria switching cost alto — cliente investiu, difícil sair
- Posiciona como parceiro estratégico, não só software
- Permite recorrente maior (R$2K/mês vs R$400–600/mês SaaS puro)

**Desvantagens:**
- Ciclo de venda mais longo (4–8 semanas)
- Requer time de implementação por cliente
- Escala mais lenta que SaaS self-serve

**Como escalar:** Criar camada de parceiros implementadores, onboarding autoguiado, 3 tiers (Basic/Pro/Enterprise)

**LTV médio projetado:** R$34K–58K por cliente (setup + 12–24 meses de recorrência).

#### Modelo C — Performance-Based

- Viabilidade: baixa como modelo principal. Clínicas resistem a compartilhar dados financeiros.
- Útil como argumento de venda: "Você paga só se gerar resultado"
- Híbrido ideal: R$800 fixo + R$50/no-show evitado acima da baseline

## Seção 7: Regulatório

### Anvisa — Software como Dispositivo Médico (SaMD)

**RDC 657/2022:** Software que realiza diagnóstico é classificado como dispositivo médico. Exige registro.

**Estratégia de produto:** Usar linguagem de "análise de aparência" e "recomendação de bem-estar" vs "diagnóstico médico". Consultar advogado regulatório antes de lançar módulo de análise de pele.

### CFM / CFBio / CRBM — Publicidade Médica

- Proibida publicidade com antes/depois para procedimentos médicos
- Vedada garantia de resultado em publicidade
- Sistema de IA DEVE conhecer essas regras — é também o diferencial de mercado

### LGPD — Dados de Saúde

- Dados de saúde são dados sensíveis (Art. 11 da LGPD)
- Termo de consentimento robusto na captura
- Penalidades: até 2% do faturamento no Brasil ou R$50M por infração

## Seção 8: Síntese Estratégica

### Posição Competitiva Atual do EsteticaFlow

- Nenhum concorrente nacional faz o mesmo com o mesmo nível de integração
- Soluções internacionais não estão no Brasil
- Soluções nacionais de gestão não têm IA de vendas/WhatsApp

**Janela de vantagem competitiva:** 12–24 meses. Usar para criar switching cost e base de dados proprietária.

**5 Decisões Estratégicas Mais Importantes:**

1. **Aprofundar antes de ampliar.** Subir ARPU de R$2K para R$3.5K com upsells nos clientes atuais antes de gastar CAC em novos clientes.

2. **SkinScore é o próximo produto standalone de maior potencial.** Alta demanda, diferenciação clara, funciona como "isca" que traz novas clínicas (demo gratuita de análise de pele como lead gen).

3. **Franquias são o maior multiplicador de receita.** 1 contrato com rede de 50 unidades = 50 contratos com 1/10 do esforço de vendas.

4. **Distribuidores de insumos são o canal mais estratégico de longo prazo.** Galderma, Galena e similares já visitam 100% do público-alvo toda semana. Co-selling ou white label = distribuição massiva sem CAC.

5. **Dados são o moat defensável.** À medida que o EsteticaFlow acumula dados de agendamento, procedimentos, resultados e comportamento de paciente de centenas de clínicas, cria-se um ativo impossível de replicar.
