## 5. Análise Competitiva — Posição do EsteticaFlow

### Vantagens Atuais (Moat)
- Único player BR com IA de vendas + WhatsApp integrado + anti-no-show especializado em estética
- Modelo de implementação cria switching cost alto (cliente investiu R$10K, não sai facilmente)
- Base de dados acumulada de leads, procedimentos e comportamento de paciente — ativo crescente

### Riscos
- Replicação por agências de automação (12–18 meses)
- iClinic ou Nuvem Software adicionando módulos de IA (12–24 meses)
- Plataformas genéricas (ManyChat, Blip) adicionando LLM nativo

### Janela de Oportunidade
**12–24 meses de vantagem competitiva clara.** Usar para:
1. Criar network effect via base de clínicas + dados proprietários
2. Fechar 1 contrato com franquia nacional (cria barreira simbólica e de base)
3. Lançar pelo menos 2 produtos adicionais que aumentem switching cost

## 6. Regulatório — Pontos de Atenção

| Risco | Impacto | Como Mitigar |
|-------|---------|--------------|
| Anvisa RDC 657/2022 (SaMD) | Módulo de análise de pele pode exigir registro | Posicionar como "análise de aparência", não "diagnóstico" |
| CFM/CFBio — publicidade médica | Produto de conteúdo deve conhecer restrições | Treinar LLM nas resoluções dos conselhos |
| LGPD — dados sensíveis de saúde | Multa até 2% do faturamento | Consentimento explícito + DPO + retention policy |
